走向视网膜时代

视网膜：显示屏的分辨率极高，使得肉眼无法分辨单个像素。首先来了解一些概念

设备像素
也称物理像素。它是显示设备中一个最小的物理部件，每个像素都可以根据操作系统设置自己的颜色和亮度。
屏幕密度
指的是一个设备表面上存在的像素数量，通常以每英寸多少像素来计算（ppi）。当屏幕密度超过300ppi是人眼 就无法分辨。平板显示器超过360ppi就不会用颗粒感。
css像素
css像素成为与设备无关的像素。在一个标准显示密度下，一个css像素对应一个设备像素。例如在pc上 例如<div width="20" height="20"></div >在pc上显示20X20像素，在retina为2 的设备上显示了40X40个像素。但是物理尺寸是一样的。这就导致了每个像素点，是普通的四个像素点。 在开发中用device-pixel-ratio来解决。
位图像素
当一个光栅图像在标准设备下全屏显示时，一个位图像素对应个设备像素，保真显示。但是在retina下他要放大 4倍莱宝石相同的物理像素的大小，这样会丢失细节造成失真。我估计模拟是
